##主要内容
###GameMode类
主要创建了游戏的胜利和失败逻辑，以及添加对应的胜利或失败UserWidget组件；添加了游戏的主界面UserWidget组件
###Character类
玩家操控的角色C++类名为AGame2Character，角色的移动为模板初始移动，后续在蓝图类中调整了镜头的位置；
